# Tokki
NewJeansのWebファンサービスです。
会員登録(Firebase Auth)をすると、各メンバーの掲示板に書き込むことができます。

# 技術
React.js / Next.js / TypeScript / Chakra UI / React Hook Form / Firebase ( Authentication, Firestore )
